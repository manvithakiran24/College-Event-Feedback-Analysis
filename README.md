# College-Event-Feedback-Analysis
College Feedback Analysis project using Python. Processes Google Forms data, extracts ratings, performs sentiment analysis, and visualizes results with charts and word clouds. Highlights top-rated events, common complaints, and department-wise performance to help improve future activities.
This project analyzes student feedback collected from Google Forms. It extracts ratings, performs sentiment analysis on comments, and visualizes key insights through charts and word clouds. The goal is to identify top-rated events, common complaints, and department-wise performance to help improve future activities.
